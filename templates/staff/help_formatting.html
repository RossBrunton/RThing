The following is all the formatting available, and what they result in.

You can divide your text into paragraphs by having an empty line between them, like so: 
This is the first paragraph

This is the second paragraph


The following syntax can be used inside paragraphs:

Code Description
$a^2 = b^2 + c^2$ Inline LaTeX
[mono]Text[/mono] Displays the text in monospace
[url]http://example.com[/url] Displays a link to the given URL
[url http://example.com]Example[/url] Displays a link to the given URL with the text "Example"
[img]1.png[/img] Displays the image that was uploaded by the manage files page named 1.png
[img]http://example.com/1.png[/img] Displays the image that is at that URL
[youtube]1234567[/youtube] Displays the Youtube video with that ID
{{ "[html]This is HTML[/html]" }} Inserts raw HTML
#1 + 1# Displays the text such that the user can click it to enter it into a prompt

The following syntax works on whole paragraphs, that is, when using them there has to be an empty line between them and the next or previous paragraph.
Code Description
$$a^2 = b^2 + c^2$$ The whole paragraph will be rendered as LaTeX, and be dispalyed in the middle of the screen
##1 + 1## Like #, but surrounds a paragraph and makes the whole paragraph clickable
Note: If it is the first word in a paragraph, that paragraph will be displayed as a note
Warning: If it is the first word in a paragraph, that paragraph will be displayed as a warning
Info: If it is the first word in a paragraph, that paragraph will be displayed as extra information
